# F5-BIG-IP-URL
一个功能强大的 F5 BIG-IP URL 编码/解码工具，提供图形用户界面，支持单个和批量处理操作。

✨ 主要功能
🔍 单个URL解码

解码 F5 BIG-IP 编码的 URL（格式：f5-w-[十六进制编码]$$）
显示详细的解码信息：

F5服务器地址
原始目标地址
附加路径
最终完整URL
十六进制编码


支持一键复制解码结果

📁 批量URL解码

从文件中批量处理多个URL
支持 .txt 格式文件输入
实时进度显示和统计信息
详细的处理报告（成功数、失败数、成功率）
支持保存批量处理结果到文件
创建示例文件功能，方便用户理解格式

🔒 URL加密功能

将普通URL转换为F5 BIG-IP格式
生成标准的 f5-w-[十六进制]$$ 格式
提供使用示例和说明

🎨 用户界面特性

现代化的图形界面，使用 Tkinter
标签页设计，功能模块清晰分离
跨平台兼容（Windows、macOS、Linux）
智能剪贴板支持
实时状态显示
进度条显示批量处理进度
